# Retail-Mart-Data-Analytics-with-Pandas-NumPy
A complete Mart / Retail sales data analysis project using Python. Includes data cleaning, preprocessing, exploratory data analysis (EDA), trend detection, customer behavior insights, and rich visualizations using Pandas, NumPy, Matplotlib, and Seaborn.


This repository contains an end-to-end Mart (Retail) Data Analysis project built using Python.
It demonstrates the full workflow of analyzing supermarket/mart sales data:
  - Data loading & cleaning
  - Handling missing/inconsistent values
  - Exploratory data analysis (EDA)
  - Sales trend forecasting
  - Customer behavior insights
  - Product-level performance analysis
  - Visual summaries & dashboards

Powered by Pandas, NumPy, Matplotlib, Seaborn, and Plotly, this project reveals patterns in sales, revenue, category performance, seasonal trends, and customer transactions.
